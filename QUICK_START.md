# MSST WebUI 安装包快速使用指南

## 🚀 快速开始

### 1. 制作安装包

```bash
# 运行构建脚本
build_installer_final.bat
```

### 2. 安装程序

1. 双击 `installer/MSST_WebUI_Setup_1.0.0.exe`
2. 按照向导完成安装
3. 选择安装路径、端口、账户等配置

### 3. 启动程序

- 双击桌面快捷方式 "MSST WebUI"
- 或使用开始菜单

### 4. 访问 WebUI

- 打开浏览器访问：http://localhost:端口号
- 使用设置的用户名和密码登录

## 📋 安装包功能清单

✅ **安装路径选择** - 用户可自定义安装目录  
✅ **临时用户目录** - 可配置上传/下载目录

✅ **初始账户设置** - 设置管理员用户名和密码  
✅ **桌面快捷方式** - 创建 MSST.bat 快捷方式  
✅ **配置管理工具** - 集成配置管理功能  
✅ **安装验证工具** - 验证安装完整性  
✅ **多语言支持** - 中文/英文界面  
✅ **完整卸载** - 支持完全卸载

## ❓ 常见问题

### Q: 安装失败怎么办？
A: 检查管理员权限、磁盘空间、防病毒软件设置

### Q: 无法启动程序？
A: 运行安装验证工具检查问题，或查看日志文件

### Q: 无法访问 WebUI？
A: 检查防火墙设置、端口配置、服务器状态

### Q: 如何修改配置？
A: 使用配置管理工具或直接编辑配置文件
