# MSST WebUI 安装说明

## 欢迎使用 MSST WebUI

MSST WebUI 是一个强大的音频分离和处理工具，支持多种音频格式和模型。

## 安装前准备

1. **系统要求**
   - Windows 10/11 (64位)
   - 至少 4GB 可用磁盘空间
   - 建议 8GB 以上内存

2. **安装权限**
   - 需要管理员权限进行安装
   - 安装程序会自动处理所有必要的配置

## 安装过程

### 1. 选择安装路径
- 默认安装路径：`C:\Program Files\MSST WebUI`
- 可以自定义安装路径

### 2. 选择组件
- **完整安装**：安装所有组件（推荐）
- **自定义安装**：选择需要的组件

### 3. 服务器配置
- **端口设置**：默认端口 7860，可自定义
- **管理员账户**：设置初始用户名和密码

### 4. 目录配置
- **输入目录**：临时输入文件存储位置
- **输出目录**：处理结果输出位置
- **缓存目录**：模型和临时文件缓存位置

### 5. 桌面快捷方式
- 可选择创建 MSST WebUI 快捷方式
- 可选择创建配置管理工具快捷方式

## 安装后使用

### 启动程序
1. 双击桌面快捷方式 "MSST WebUI"
2. 或在开始菜单中找到 "MSST WebUI"

### 配置管理
- 使用配置管理工具修改设置
- 支持修改端口、目录、用户等配置

### 访问 WebUI
- 服务器地址：http://localhost:端口号
- 客户端地址：http://localhost:7861

## 重要说明

1. **首次运行**
   - 程序会自动创建必要的目录
   - 首次启动可能需要较长时间

2. **防火墙设置**
   - 如果无法访问，请检查防火墙设置
   - 确保相应端口未被阻止

3. **目录权限**
   - 确保输入输出目录有读写权限
   - 建议使用非系统盘目录

4. **模型文件**
   - 首次使用需要下载模型文件
   - 请确保网络连接正常

## 卸载程序

1. 通过控制面板卸载
2. 或使用开始菜单中的卸载程序
3. 卸载时会保留用户数据（可选）
